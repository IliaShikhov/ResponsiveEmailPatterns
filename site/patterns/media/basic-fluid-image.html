Basic Fluid Image

The simplest tactic for dealing with responsive images. The image is set to 100% of the width of it's container simply scales down as the screen size shrinks.

            
                        <style>
	/* Basic Fluid Image Pattern CSS */
    @media only screen and (max-width: 599px) {
        img.pattern {
            width: 100%;
            height: auto !important;
        }
    }
</style>
